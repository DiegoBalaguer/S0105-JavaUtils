====================================================================================================
EJERCICIOS - TAREA S01.05: JAVA UTILS
====================================================================================================

Todos los ejercicios han sido desarrollados utilizando el IDE IntelliJ, con la versión 21 del
JDK de Oracle Java, que tiene que estar correctamente instalada en el equipo cliente para que
el proceso funcione.

Para facilitar su ejecución desde la consola, se ha preparado un archivo por lotes llamado
"S0105-JavaUtils-runner.bat", que se encarga de compilar y ejecutar los distintos ejercicios
de forma secuencial.

Durante la compilación, se genera un archivo intermedio "sources.path" que contiene la lista de
archivos ".java" correspondientes a cada ejercicio. Este archivo se utiliza para compilar mediante
el comando "javac", utilizando el parámetro "-d" para indicar la ruta de salida de los archivos
".class", que es en la carpeta "outJava" que cuando finaliza la ejecución se borra.

Para la ejecución, se emplea el parámetro "-cp" para especificar la ruta en la que se encuentran
los archivos ".class".

**Importante:** el proceso se tiene que ejecutar desde la raíz del proyecto "/S0105-JavaUtils".

Como confirmación visual de que cada módulo se ha ejecutado correctamente, se muestra al final de
cada ejercicio el siguiente mensaje:

✅ Process completed correctly
